# دفتر الديون - نسخة خفيفة

تطبيق أندرويد Flutter عربي خفيف لإدارة الديون والتسديدات بدون إنترنت.

## المزايا
- إضافة عميل / مدين.
- تسجيل دين.
- تسجيل تسديد.
- كشف حساب لكل عميل.
- بحث.
- ملخص إجمالي.
- حفظ محلي داخل الهاتف.
- لا يحتاج صلاحيات.
- بناء APK عبر GitHub Actions.
- محاولة ضبط minSdk على 21 ليعمل على أجهزة قديمة كثيرة.

## البناء
ارفع الملفات إلى GitHub ثم شغل:
Actions -> Build Android APK
